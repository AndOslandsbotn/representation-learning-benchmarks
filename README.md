# representation-learning-benchmarks
This repository contains benchmarks on different architectures used for capturing the intrinsic structure of high-dimensional data.

We are studying how different architectures reshape extrinsic geometry into latent manifolds.

The geometric spaces we consider are
1. Extrinsic space
- Pixel space
- $\mathbb{R}^D$

2. Latent space
- Model embedding
- $\mathbb{R}^d$

3. Visualization space / Projection
- t-SNE / Laplacian eigenmaps / Voltage maps
- $\mathbb{R}^{\{2,3\}}$
